# Directives (reviewer → implementer)

Reviewer agents write `DIRECTIVES-YYYY-MM-DD.md` here.

- Read-only analysis only when producing these files.
- Implementer applies to code (`src/lib/prompts.ts`, schema soft-checks, etc.) **after owner approval**, then updates live docs under `docs/product.md` / `reference.md`.
- These files stay as **history**; do not treat them as current product spec alone.

See `../HANDOFF-REVIEWER.md` for the handoff prompt and file list.
